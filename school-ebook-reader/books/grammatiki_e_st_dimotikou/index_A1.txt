            γεγονότα ή περιγράφουμε 
            πρόσωπα, ζώα και πράγματα.
          

            Αλλά με τη γλώσσα μπορώ επίσης να κάνω 
            πράγματα: να καλέσω τους φίλους μου, να
τους ευχηθώ, να υποσχεθώ κάτι στη μαμά 
            μου, να δώσω συμβουλές στη μικρή μου

            μπαίνει στο δωμάτιο μου χωρίς να με 
            ρωτήσει. Κάθε φορά που μπαίνει φέρνει 
            καταστροφή!

            ΔΡΑΣΤΗΡΙΟΤΗΤΑ 2
          

            Συζητώ στην τάξη και βρίσκω τη λεκτική πράξη που ταιριάζει σε 

            κάθε περίπτωση. Μετά τη γράφω στο καρτελάκι της κάθε εικόνας.
          

            Απαγόρευση, Συμβουλή, Υπόσχεση, Δήλωση, Ευχή, Παράκληση, Διαταγή, Έπαινος
          

            ΔιαταγήΤι θα θυμάμαι για... τις λεκτικές πράξεις



Χρησιμοποιώ τη γλώσσα όχι μόνο για να περιγράψω πράγ-
                ματα ή να αφηγηθώ γεγονότα, αλλά και για να κάνω διά-
                φορες πράξεις. Αυτές ονομάζονται λεκτικές πράξεις.
Μπορώ να ευχηθώ, να υποσχεθώ και να δηλώσω κάτι.
                Μπορώ ακόμα να δώσω συμβουλές, να ζητήσω κάτι, να 
διατάξω ή να απαγορεύσω. Μπορώ επίσης να εκφράσω
τη διάθεσή μου για κάποιον ή κάτι (συμπάθεια, έπαινο
                κτλ.).
Τις περισσότερες φορές, στις λεκτικές πράξεις χρησιμο-
                ποιώ συγκεκριμένους γραμματικούς τύπους (π.χ. οριστι-
                κή, υποτακτική, προστακτική έγκλιση, μέλλοντα χρόνο)
                ή συγκεκριμένα ρήματα (π.χ. υπόσχομαι).
