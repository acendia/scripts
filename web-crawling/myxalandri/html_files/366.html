Το Πασχαλινό Μπαζάρ με ελληνικές επιχειρήσεις που δεν πρέπει να χάσεις!

mymag

Πλησιάζει το Πάσχα και οι μέρες που θα βγούμε για δώρα και αγορές έρχονται! Φέτος λοιπόν καθώς ψάχναμε λαμπάδες, πέσαμε πάνω σε ένα αγαπημένο μας κατάστημα που εδρεύει εδώ στο Χαλάνδρι, το Kristia Lazarie, που πήρε μια υπέροχη πρωτοβουλία να διοργανώσει Pop Up Market!

Θα πραγματοποιηθεί το Σαββατοκύριακο 1-2 Απριλίου στον εκθεσιακό χώρο του ''Τέρας'' στον Νέο Κόσμο (στην οδό Θεοδώρητου Βρεσθένης 45). Σκοπός του είναι να γνωρίσουμε και να στηρίξουμε μικρές τοπικές επιχειρήσεις.

Πρόκειται να στηθεί για 2 μέρες μια γιορτινή αγορά. Ταλαντούχοι εκθέτες ενώνονται σε ένα μοναδικό event με ιδιαίτερα προϊόντα και μεγάλη ποικιλία. Έτσι λίγες μέρες πριν το Πάσχα θα βρείτε 20 brands μαζεμένα σε ένα μέρος, καθένα με διαφορετικά είδη, κατασκευασμένα στο χέρι με πολύ μεράκι και φαντασία! Brands έτοιμα να σας ξεναγήσουν και να δώσουν λύσεις και ιδέες για δωράκια.

Ταυτόχρονα την 1η μέρα του event τα Braids by Emma θα είναι εκεί για να κάνουμε τα πιο απίθανα box braids πλεξουδάκια στα μαλλιά. 

Στην εκδήλωση θα υπάρχουν πολύχρωμες αρωματικές λαμπάδες (που όσοι μένουμε Χαλάνδρι μπορούμε να δούμε και στο φυσικό κατάστημα Kristia Lazarie παρέα με τα αρωματικά κεριά τους). Θα βρείτε χειροποίητα κοσμήματα και καλοραμμένα ρούχα από ανοιξιάτικες συλλογές! Vintage αισθητικής γυαλιά ηλίου, τσάντες & αξεσουάρ. Θα γνωρίσετε παραγωγούς vegan και eco friendly καλλυντικών. Σίγουρα θα μπείτε στο πειρασμό να ψωνίσετε από τα πρωτότυπα κεραμικά για το σπίτι σας ή τα φυτά εσωτερικού χώρου που ζούνε σε μικρές διάφανες γυάλες. Άλλη μια όμορφη ιδέα για δώρο! 

Επίσης θα ικανοποιήσετε και τις ανάγκες των τετράποδων φίλων σας αφού στο χώρο θα δείτε μια boutique κατοικιδίων με όλα όσα χρειάζονται!

Γνωρίστε τους εκθέτες μας!
